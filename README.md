# Влияние контентных эмбеддингов на рекомендации

## О проекте

Данный Jupyter Notebook реализует сравнительные исследования различных типов контентных эмбеддингов 
на качество рекомендаций в задаче коллаборативной фильтрации.

Проект сравнивает:

1. Чисто коллаборативные подходы (Popularity, SVD, UserKNN)
2. Контентные модели (текстовые, визуальные, мультимодальные)
3. Гибридные модели (UserKNN + контент)

## Основные цели
1. Оценить вклад разных типов эмбендингов (текст, визуал, мультимодальность, метаданные)
2. Изучить эффективность гибридизации
3. Проанализировать устойчивость моделей к шуму и случайным начальным условиям

## Датасет
Использован датасет **VK-LSVD** (subsample `up0.001_ip0.001`)

Период обучения: 0 - 4 недели

Валидация: 25 неделя

Положительные взаимодействия определяются по времени просмотра ≥ 30 сек + лайки, репосты, закладки и т.д.

## Используемые библиотеки

- `polars`, `pandas`, `numpy`, `scipy`
- `scikit-learn` (SVD, TF-IDF, cosine similarity)
- `sentence-transformers` (SBERT)
- `huggingface_hub`
- `matplotlib` + `seaborn`

## Реализованные модели

### Базовые
- **Popularity Baseline**
- **SVD** (матричная факторизация)
- **UserKNN** (пользователь - пользователь)

### Контентные
- **TF-IDF** (суррогатные текстовые признаки)
- **SBERT** (семантические эмбеддинги)
- **Visual** (визуальные эмбеддинги через SVD)
- **Multimodal** (комбинация визуальных + текстовых)

### Гибридные
- UserKNN + Visual (α = 0.5)
- UserKNN + Multimodal (α = 0.25)

## Результаты

Добавление визуальных контентных эмбеддингов даёт наибольший прирост качества рекомендаций (+~53% NDCG@10 по сравнению с бейзлайном и +53% по сравнению с чистым UserKNN).

Гибридная модель UserKNN + Visual демонстрирует лучшее сочетание точности, качества ранжирования и устойчивости к шуму и случайным факторам.
